# Moon Ceramic - Furniture Store Website

A responsive furniture eCommerce website built from a Figma design using HTML, CSS, Bootstrap, and JavaScript. The project focuses on creating a modern shopping experience with a clean interface and responsive layouts.

## Features

- Responsive design
- Home, Shop, Product, Cart, Checkout, About, Blog, and Contact pages
- Product categories
- Newsletter section
- Interactive UI with JavaScript
- Clean and organized code

## Technologies

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Font Awesome
- Google Fonts

Clone the repository:

```bash
git clone https://github.com/ziadmohamedali/Ecommerce-moon.git
```
