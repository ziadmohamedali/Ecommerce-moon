# Moon Ceramic - Furniture Store Website

A modern, responsive furniture eCommerce website built from a detailed Figma design. This project serves as the semi-final milestone for the Eraasoft online web development course.

While the project guidelines originally required only HTML, CSS, and Bootstrap, I wanted to push my boundaries and make my work stand out. Despite being only three weeks into learning JavaScript in our online sessions, I jumped ahead of the syllabus to self-teach and build out custom JS functionality for each individual page.

---

## Features

* Complete Multi-Page Architecture: Features dedicated pages for Home, Shop, Product Details, Cart, Checkout, About, Journey, Blog, and Contact.
* New Journey Page: A dedicated page (Journey.html) paired with custom styling (Journey.css) to highlight the brand's timeline and story.
* Pixel-Perfect Implementation: Translated a professional, modern Figma interface design into clean, responsive layouts.
* Dynamic JavaScript Functionality: Self-taught and implemented targeted JS files for page-specific interactive logic, including Cart rendering, Checkout validations, and UI state management.
* Responsive Layouts: Crafted to work across mobile, tablet, and desktop viewports using Bootstrap 5 and custom CSS.

---

## Tech Stack Used

* HTML5: Semantic document markup.
* CSS3: Custom layouts, responsive styling, and UI transitions.
* Bootstrap 5: Fluid grid systems and base styling components.
* JavaScript (ES6+): Scripting applied ahead of the course syllabus to introduce dynamic user experiences.
* Font Awesome: Iconography.
* Google Fonts: Custom typography.
